Java基础

对象相等和对象引用相等的区别

对象的相等比较的是存放内容是否相等。

引用的相等比较的是指向的地址是否相等。

深拷贝和浅拷贝的区别

深拷贝和浅拷贝主要在一个对象内部的属性是引用对象时有区别。浅拷贝的拷贝对象和原对象公用一个内部对象，而深拷贝会复制整个对象，包括内部对象。

抽象类和接口

共同点：

  • 都不能被实例化
  • 都可以有抽象方法
  • 都可以由默认实现

区别：

  • 接口是一种约定，一个类实现了一个接口，就表明这个类实现了相关接口的方法。抽象类主要用于复用代码。
  • 接口中的属性访问权限是public static final 的。抽象类的属性访问权限默认是default的。
  • 一个类只能继承一个父类，但是可以实现多个接口。

重载和重写区别

重写和重载都是方法名字相同的函数，不过重写发生在父子类中，重载发生在同一个类中。

方法名：必须相同

参数列表：重载必须不同，重写必须相同

访问权限：重载可以不同，重写只能更大

抛出的异常：重载可以不同，重写只能更小

返回类型：重载可以不同，重写只能更小

发生阶段：重载在编译器，重写的运行期

包装类型和基本类型的区别

空间占用上：包装类型需要的空间比基本类型大

默认值上：包装类型没有默认值，没有复制的包装类型是null，基本类型有默认值

比较方式上：包装类型使用==进行比较的时候，是比较引用相等，所以比较包装类型的时候，需要使用 equals 方法进行比较

用途：包装类型可以用于泛型，但是基本类型不可以，基本类型常常会用于局部变量和常量，包装类型常常用于方法参数和成员变量

包装类型的缓存机制

包装类型的缓存机制是用于在自动装箱的时候，尽可能提高创建对象的效率而做出的缓存行为。

Byte、Short、Integer、Long 四种类型默认缓存了 [-128, 127] 相应的对象，而 Character 类型默认缓存了 [0, 127] 范围的对象。

语法糖

语法糖是为了方便程序员开发而进行设计的一种特殊语法，常见的语法糖有：泛型，自动拆装箱，变长参数，增强for循环等。JVM并不能识别语法糖，所以语法糖需要在编译阶段进行解糖操作。

HashMap

hashmap的主要作用是以平均时间复杂度为O(1)来进行元素的高效存取，在jdk1.8中，hashmap的底层是由hash桶加链表和红黑树来进行实现的。

当hash桶长度大于等于阈值并且链表长度大于等于阈值的时候，会将链表转为红黑树，但是只有当链表长度退化到退化阈值的时候，才会进行树转化为链表的操作。

hash桶的长度总是2的幂次方数。也就是说，如果你初始化的时候传入的容量是30，那么在hashmap第一次添加元素的时候，会将hash桶的长度设置为32。
